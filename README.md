# 个人私有云仓库

一个面向单人使用的私有文件仓库 Web MVP，支持：

- 手机、电脑上传
- 图片 / 视频 / 文档 / 文本 / 其他文件
- 按时间、类型、名称、大小浏览
- 在线预览图片、视频、文本、PDF
- 备注、标签、收藏
- 回收站
- 批量下载
- 分块上传与失败重试

## 启动

```bash
npm start
```

默认地址：

```text
http://127.0.0.1:3000
```

## 存储位置

- 文件内容：`storage/files`
- 临时分块：`storage/tmp`
- 元数据：`data/vault.json`
- 上传会话：`data/uploads.json`

## 说明

- 这是一个单用户个人仓库，不包含显式登录界面。
- 默认建议部署在私有网络、VPN 或反向代理之后。
- `PPT/PPTX` 当前以下载为主，图片、视频、文本、PDF 可直接预览。
- 批量下载会按选中顺序逐个触发浏览器下载。
